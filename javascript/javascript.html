JavaScript

Indentificadores

  • Podem começar com letra, $ ou _
  • Não podem começar com números
  • É possivel usar letras ou numeros
  • É possibel usar acentos e símbolos
  • Não podem conter espaços
  • Não podem ser palavras reservadas

Tipos Primitivos

  • Number
    • Infinity
    • NAN
  • String
  • boolean (true ou false)
  • undefined
  • Null

Operadores Aritmeticos

5 + 2 = 7 Adição

5 - 2 = 3 Subtração

5 * 2 = 10 Multiplicação

5 / 2 = 2.5 Divisão

5 % 2 = 1 Resto da divisão

5 ** 2 = 25 Potencia

Ordem De Precedência

  1. ()
  2. **
  3. * / %
  4. + -

Auto-Atribuição

Var n = 3

n = n + 4

7

simplificando

Var n = 3

n += 4

7

Incremento

Var n = 3

n++

4

Relacionais

  • > maior
  • < menor
  • >= maior ou igual
  • <= menor ou igual
  • == igual (valor)
  • === idêntico (valor e tipo)
  • != não igual (valor)
  • !== não igual (valor e tipo)

Operadores lógicos

  • ! negação quero uma caneta azul não vermelha
  • && conjunção quero uma caneta azul e vermelha
  • || disjunção quero uma caneta azu ou vermelh

Ordem De Precedência

  1. () ** / ...
  2. > > >= ...
  3. !
  4. &&
  5. ||

Condição

Condição (if / else)

if (idade >= 18) {
console.log("maior de 18")
} else {
console.log("menor de 18")
}

Explicação

Ele verifica uma condição (verdadeira ou falsa) e executa um bloco de código apenas se a condição for verdadeira.

Condição Aninhadas

if (idade >= 18) {
console.log("maior de 18")
} else if (idade >= 50){
console.log("senhor de idade")
} else {
console.log("menor de idade")
}

Explicação

Ele verifica uma condição (verdadeira ou falsa) e executa um bloco de código apenas se a condição for verdadeira.
