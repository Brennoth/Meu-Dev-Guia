Identificadores

  • Podem começar com letra ou _
  • Não podem começar com número
  • É possível usar letras ou números
  • É possível usar acentos e símbolos
  • Não podem conter espaços
  • Não podem ser palavras reservadas

Tipos primitivos

  • int: numeros inteiros ex: 7, -4, 0
  • floot: numeros reais ex: 4.5 | 0.076
  • bool: valores logicos ex: true,false
  • string: textos ex: "ola", "7.5"

Operadores Aritmeticos

  • + Adição ex: 5+2 = 7
  • - Subtração ex: 5-2 = 3
  • * Multipliacação ex: 5*2 = 10
  • / Divisão ex: 5/2 = 2.5
  • ** Potência ex: 5**2 = 25
  • // Divisão inteira ex: 5//2 = 2
  • % Resto da divisão ex: 5%2 = 1

Ordem de procedencia

  1. ()
  2. **
  3. * / // %
  4. + -

Dicionario

Dicionario { }

aluno = {"nome": "brenno",
"idade": 20,
"cidade": "RJ"
}

Explicação:

Um dicionário é uma coleção não ordenada.
Os elementos são guardados em pares chave-valor.
A chave funciona como um identificador único.
O valor é o dado que você quer guardar.
A vantagem é que você não precisa lembrar a posição (índice), só a chave.

Use dicionário quando cada dado precisa de uma “etiqueta” (chave) para ser encontrado rapidamente.

Códigos

Explicação:
